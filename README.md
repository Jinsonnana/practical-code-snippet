## 项目说明 ##

运行本项目，需要全局安装VuePress：

``` Shell
# 安装
yarn global add vuepress # 或者：npm install -g vuepress

# 新建一个 markdown 文件
echo '# Hello VuePress!' > README.md

# 开始写作
vuepress dev

# 构建静态文件
vuepress build
```

构建完成之后上传到github就可以了。
